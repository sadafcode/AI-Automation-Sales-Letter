---

## 1. PROBLEM

❌ Every booking required a manual phone call  
❌ Finding the right provider took hours  
❌ No price visibility before booking  
❌ No proper system for complaints or disputes  
❌ Literacy barrier — not every user can type  
❌ Service only available during business hours  

---

## 2. SOLUTION — 9-AGENT MULTI-AGENT ORCHESTRATION

We built the world's first **9-Agent AI Orchestration System** for home services.  
Not one AI — **9 specialized agents** working together like a professional team.

✅ Each agent has one dedicated job  
✅ Master Orchestrator coordinates all agents in real time  
✅ Zero manual steps — fully automated end-to-end  
✅ Understands Urdu and Roman Urdu naturally  
✅ Tap-to-book UI — no typing required  
✅ Available 24/7 — no missed opportunities

## 4. ALL 9 AGENTS — WHAT EACH ONE DOES

| # | Agent Name | Role |
|---|-----------|------|
| 1 | **NLU Agent** | Understands Urdu & Roman Urdu text input |
| 2 | **Intent Agent** | Detects what service the user needs |
| 3 | **Orchestrator Agent** | MASTER — coordinates all other agents |
| 4 | **Discovery Agent** | Finds nearest, best-rated provider via GPS |
| 5 | **Pricing Agent** | Estimates a fair price instantly |
| 6 | **Booking Agent** | Confirms provider, books slot, notifies customer |
| 7 | **Feedback Agent** | Collects review & rating after service |
| 8 | **Dispute Agent** | Receives and processes complaints automatically |
| 9 | **Dispute Report Agent** | Generates a detailed complaint resolution report |
